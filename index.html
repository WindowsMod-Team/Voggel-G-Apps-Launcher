Voggel G-Apps Launcher
| v1.0.0

Alle Google Apps an einem Ort.

Alle Google Apps in einem Launcher, klicken sie eine an und sie öffnet sich und kann him hintergund weiter laufen (Minimiert werden). Zudem kann sie den Vollbild Modus Aktivieren. Wenn sie Fertig sind, dann kann die App einfach geschlossen werden.

Voraussetzungen:

• Microsoft Edge Version: Aktuelle Version von Microsoft Edge.
• Windows System: Es wird ein 64-Bit System Benötigt.
• Speicherplatz: Es sollten 1GB freier Speicherplatz verfügbar sein, da die App 1,36MB benötigt.
